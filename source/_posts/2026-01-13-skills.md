## skills

**🛠️ Claude Skills - 賦予 AI 助理與現實世界互動的能力**

`Claude Skills` 是 Anthropic 為其模型（特別是 Claude 3.5 Sonnet）所定義的一套擴展能力，允許模型不再僅限於文字生成，而是能透過 **工具調用 (Tool Use)** 與外部世界互動。

其核心邏輯在於：「當模型發現自己無法獨立完成任務時，主動要求執行特定的程式碼或 API。」🤖✨

<!-- more -->

# 🌟 核心概念
- **工具調用 (Tool Use / Function Calling)**：讓模型能呼叫預定義的函數來執行計算、查詢資料或操作系統。
- **電腦操作能力 (Computer Use)**：讓 Claude 具備操作 Windows GUI 或 PowerShell 的能力，進行複雜的環境調適。
- **模型上下文協定 (Model Context Protocol, MCP)**：標準化的資料交換格式，輕鬆整合 API 或資料庫。

# 💡 重點摘要 (Key Summary)
- **從被動到主動**：具備 Skills 的 Claude 會根據目標尋找工具，並自行決定執行的先後順序。
- **生態系整合**：透過 MCP，可以將後端 API 或資料庫結構封裝成 Skill，讓 AI 直接查詢或執行腳本。
- **環境感知**：使 AI 能夠進行自動化環境設定或 UI 測試。
- **循環反饋機制**：執行失敗時，AI 會根據錯誤訊息自動修正參數並重新嘗試。

# 🗂️ 專案技能清單 (Project Skills)
本專案目前收錄並維護多項專用技能，包含：
- 🔍 **Code Reviewer**：專業的代碼審查與建議。
- 📝 **Obsidian Markdown/Bases**：深度整合 Obsidian 筆記系統。
- 🎨 **JSON Canvas**：操作與管理視覺化畫布。
- 🛠️ **Debug Wizard**：自動化的除錯與問題診斷。
- 📊 **Log Sentinel**：監控與分析系統日誌。
- 🎨 **Manga Master**：趣味性的 AI 漫畫創作輔助。

# 🏗️ 專案宗旨
`skills` 專案旨在建立一套標準化且可擴展的 AI 工具庫，讓 AI 助理從「語義理解」跨越到「實際執行」，成為真正的 AI 生產力夥伴。
