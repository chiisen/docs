## 波妞消消樂 (Match-3 Game) 🐾🐱✨

**✅ 一款以可愛貓咪（波弟）與可愛小女生（波妞）為主題的網頁版三消益智遊戲。**

這是一款結合了可愛貓咪與溫馨主題的經典三消遊戲。採用 HTML5 Canvas 渲染技術，不論是在電腦桌機還是各類行動裝置上，都能提供極致流暢的消除體驗。快來與波弟和波妞一起挑戰最高分吧！

<!-- more -->

# 🎮 遊戲特點
- 🐈 **雙角色主題**：結合可愛貓咪（波弟）與可愛小女生（波妞）的精美圖示，包含貓臉、肉球 🐾 與毛線球等元素。
- ⏳ **雙模式選擇**：
    - **經典模式**：無時間限制，輕鬆享受連鎖消除的快感。🧘‍♂️
    - **計時模式**：在 60 秒內挑戰最高分，測試你的極限反應速度！🔥
- 📱 **手機完美優化**：支援流暢的觸控滑動 (Swipe) 手勢，具備聰明的響應式佈局，自動填滿各種螢幕空間。
- 🎵 **動態音效**：內建 Web Audio API 音效，隨連鎖次數提高音調，提供沉浸式的聽覺回饋。🎧
- ⚡ **極致效能**：針對行動裝置深度優化渲染流程，確保在各種裝置上都能穩定 60 FPS 運行。
- 🎬 **互動慶祝**：消除成功時，畫面下方的貓咪影片會興奮跳轉，與你一同慶祝！🥳🐱

# 🛠️ 技術說明
- **Core**: Vanilla JavaScript (ES Modules)
- **Rendering**: HTML5 Canvas API (高效能畫布渲染)
- **Styling**: CSS3 (動態漸層與 RWD 響應式設計)
- **Audio**: Web Audio API (無延遲動態音效)
- **Deployment**: Cloudflare Pages / Workers

# 🏗️ 專案宗旨
`波妞消消樂` 旨在展示如何利用原生 Web 技術打造出具備「打磨感 (Game Juice)」的遊戲。透過連鎖音效、動態慶祝影片以及流暢的 Canvas 動畫，讓簡單的三消玩法也能擁有豐富的層次感與趣味性。
