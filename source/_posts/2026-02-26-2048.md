## 2048 網頁版 🎮

**✅ 經典 2048 遊戲的網頁版實現，支援鍵盤與觸控操作。**

2048 是一款風靡全球的益智類數字拼接遊戲。玩家的目標是透過滑動方塊，讓相同的數字相撞合併，最終挑戰達成 2048 這個神奇的數字！這款網頁版實現了流暢的動畫與直觀的操作，讓你隨時都能享受腦力激盪的樂趣。

<!-- more -->

# ✨ 特色功能
- 🧊 **流暢操作**：支援電腦鍵盤方向鍵與手機觸控滑動 (Swipe) 操作。
- 🌈 **動態視覺**：包含方塊移動的 CSS 平滑過渡、合併時的縮放彈跳效果。
- 🎨 **視覺回饋**：方塊顏色隨數字增加而加深，達成目標時更有 `canvas-confetti` 慶祝特效。
- 📱 **響應式設計**：自動適配手持裝置與桌面環境，提供一致的遊戲體驗。
- 🏁 **遊戲機制完整**：具備高分紀錄、方塊隨機生成 (2 或 4) 與遊戲結束自動偵測。

# 🛠️ 技術說明
- **Framework**: 原生 HTML5 / CSS3 / JavaScript (Vanilla JS)
- **Animation**: CSS Transitions & Keyframes
- **Effects**: `canvas-confetti` (合併成功慶祝效果)
- **Typography**: Fredoka (Google Fonts)
- **Deployment**: Cloudflare Workers

# 📖 遊戲玩法
1. **目標**：合併相同數字的方塊，最終湊出 2048。
2. **操作**：使用方向鍵 ↑ ↓ ← → 或在螢幕上滑動來移動方塊。
3. **機制**：所有方塊會朝移動方向靠攏，相同數字碰撞後會合併成兩倍數值。每次移動後會隨機產生一個新方塊。

# 🏗️ 專案宗旨
`2048` 網頁版展示了如何不依賴大型重量級框架，僅利用 CSS 動畫與原生 JavaScript 邏輯，就能打造出高性能且具備現代 UI 質感的休閒遊戲。
